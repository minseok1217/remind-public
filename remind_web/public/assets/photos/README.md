# 사진 폴더 구조

이 폴더에 사진을 추가하여 환자와의 통화에서 사용할 수 있습니다.

## 폴더 구조

```
photos/
├── memories/     - 추억 관련 사진 (여행, 가족 행사 등)
├── family/       - 가족 사진
├── activities/   - 일상 활동 사진 (산책, 취미 등)
├── nature/       - 자연 사진 (꽃, 경치 등)
└── places/       - 장소 사진 (집, 공원, 카페 등)
```

## 사진 추가 방법

1. 위 카테고리 폴더 중 하나선택
2. 사진 파일을 추가 (JPG, PNG, WebP 등)
3. 파일명은 자유롭게 설정 (예: family_photo.jpg, dog.png)

## 예시

- `memories/vacation_2023.jpg`
- `family/grandchild.jpg`
- `activities/morning_walk.jpg`
- `nature/cherry_blossoms.jpg`
- `places/our_home.jpg`

## 사진 랜덤 표시

앱에서 통화 중일 때, 이 폴더의 사진들이 랜덤하게 선택되어 표시됩니다.
